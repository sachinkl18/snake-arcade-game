# 🐍 Snake Arcade Game (Advanced Canvas JS)

## Features
- Smooth movement
- Collision detection (wall + self)
- Score + High Score
- Speed increase system
- Pause / Reset controls
- Keyboard support

## Run Locally
Just open `index.html` in browser.
